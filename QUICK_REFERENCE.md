## üì± How to Use New Features

### Setting Custom Day Reminders
1. Go to Progress tab
2. Tap "Add Custom Activity"
3. Enter activity name
4. Set time
5. Choose "Custom Days"
6. Enter number of days (e.g., 3 for every 3 days)
7. Tap "Add"

### Managing Notifications
1. Tap notification bell icon (top right)
2. View all notifications
3. Tap a notification to mark as read
4. Or tap "Mark all as read"

### Viewing Recipe Details
1. Go to Feed tab
2. Tap any recipe card
3. Tap "More Details" button
4. Scroll to see all information

### Daily Check-in
1. Go to Progress tab
2. Find Daily Check-in card at top
3. Enter weight (optional)
4. Tap "Complete Daily Check-in"
5. Earn XP!

## üêõ Troubleshooting

### Notification red dot not showing?
- Make sure you have unread notifications
- Red dot only appears for new notifications

### Custom reminder not saving?
- Check notification permissions
- Ensure time is set
- Verify custom days is between 1-365

### Recipe details not loading?
- Check internet connection
- Verify Firebase is connected
- Some recipes may have incomplete data

### Layout looks weird?
- Restart the app
- Check device screen size
- Report if issue persists
